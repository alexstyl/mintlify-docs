# Image

Displays an image.

## Image Properties

### Image

|    |                                                                     |
|-------------------------|-----------------------------------------------------|
| **Image Source**        | The URL or file path of the image to be displayed.  |
| **Content Description** | A text description of the image for accessibility.  |
| **Content Scale**       | Defines how the image is resized to fit its bounds. |
